# Heart Disease Prediction System

A machine learning based web application that predicts the likelihood 
of heart disease using clinical input parameters.

The application is built using Streamlit and trained using 
classification models from Scikit-learn.

---

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- NumPy
- Pandas

---

## Features
- User-friendly web interface
- Real-time prediction
- ML model integration
- Clinical parameter input validation

---

## Model Details
The model was trained using structured healthcare data 
and achieved approximately 85% accuracy after preprocessing 
and feature selection.

---

## How to Run Locally

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

---

## Project Structure

- app.py → Streamlit frontend
- heart_disease_model.sav → Trained ML model
- heart.ipynb → Model training notebook
